# Discord Music Bot with Autoplay

A feature-rich Discord music bot with YouTube autoplay recommendations, TTS announcements, and slash commands.

## Features

- 🎵 Play songs from YouTube (search or direct links)
- 🔄 Automatic recommendation system when queue ends
- 🔊 Text-to-Speech song announcements
- ⏭ Skip tracks, view queue, and stop playback
- 🔁 Toggleable autoplay system
- 🎙 Clean audio quality with FFmpeg
